Flask	The web micro-framework used to serve the application and handle HTTP requests (uploads/results).	Required for the web implementation.
opencv-python	The core library for all image processing, including grayscale conversion, blurring, and the final sketch blend (cv2.cvtColor, cv2.GaussianBlur, cv2.divide, etc.).	The main dependency for your pencil_sketch.py.
numpy	A dependency often used implicitly by opencv-python for handling image data (arrays) and is generally considered a best practice to include for image manipulation projects.	OpenCV is built on top of NumPy arrays.
